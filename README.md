# 局部放大图工具

一款纯前端的图片局部放大标注工具，无需安装，打开浏览器即可使用。适用于论文插图、技术文档、产品展示等场景，帮助读者聚焦图片关键细节。

## ✨ 主要功能

### 🖼️ 图片上传
- 支持拖放或点击上传 PNG / JPG / SVG / WebP 格式图片
- 上传后可随时移除并更换图片

### 🔲 选取形状
- **矩形** — 标准矩形选区
- **圆角矩形** — 可自定义圆角半径
- **椭圆** — 椭圆形选区
- **圆形** — 正圆选区

### 📐 选取模式
- **自由选取** — 在图片上拖动鼠标，自由框选任意区域
- **固定大小** — 设定宽高（px），点击图片放置选区
  - 支持**相对位置定位**：勾选后输入 X / Y 坐标（以原图左上角为原点），精确定位选取框左上角

### 🔍 放大设置
- 放大倍数：1.5× ~ 10×，滑块实时调节
- 圆角半径：仅对圆角矩形选区生效

### 📌 放置方式
- **原图外** — 放大图放置在原图的上/下/左/右/对角等 8 个方向，可调节与原图间距
- **原图内** — 放大图放置在原图内部的左上/右上/左下/右下/居中位置，或自定义拖动定位

### ✋ 拖拽交互
- **拖拽选取框** — 鼠标悬停在已确认的选取框上，光标变为 ✋，拖动可移动选区位置
- **拖拽放大框** — 鼠标悬停在放大图上，拖动可自定义放大图的显示位置
- 拖拽后连线自动智能重新匹配

### 🔗 智能连线
- 自动在选取框与放大框之间绘制两条连接线
- 根据两者相对方位智能选择最优角点配对，避免交叉
- 当放大图覆盖连线时，自动隐藏连线并显示提示文字

### 🎨 样式自定义
- 选取框边框：颜色可选 / 可关闭
- 放大图边框：颜色可选 / 可关闭，线宽可调
- 连接线：颜色可选 / 可关闭，线宽可调

### 📥 导出
- 一键导出为 PNG 图片，白底高清
- 导出前可预览效果

## 🚀 使用方法

1. 直接在浏览器中打开 `inset-zoom.html`（无需服务器，纯本地运行）
2. 拖放或点击上传一张图片
3. 选择选取形状和模式，在图片上框选要放大的区域
4. 调节放大倍数、放置方式、连线样式等参数
5. 点击 **✓ 确认选取** 添加放大区域（可添加多个）
6. 如需调整，可直接拖拽选取框或放大框
7. 点击 **📥 导出PNG** 保存最终结果

## 💡 操作技巧

| 操作 | 说明 |
|------|------|
| 新建选区 | 在空白区域拖动鼠标（自由模式）或点击（固定模式） |
| 移动选区 | 悬停在选取框上出现 ✋，拖拽移动 |
| 移动放大图 | 悬停在放大框上出现 ✋，拖拽移动 |
| 撤销 | 点击工具栏 **↩ 撤销** 按钮 |
| 删除单个 | 点击放大区域列表中对应的 ✕ 按钮 |
| 清除全部 | 点击 **清除全部** 按钮 |
| 移除图片 | 点击工具栏 **🗑 移除图片** 按钮 |



## 🔧 兼容性

- 现代浏览器均可使用（Chrome / Firefox / Safari / Edge）
- 纯前端实现，无需安装任何依赖
- 图片数据不会上传到任何服务器，完全在本地处理
